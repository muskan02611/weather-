# weather-
A responsive Weather Application built using HTML, CSS, and JavaScript that provides real-time weather information for any city using the WeatherAPI. Users can search for locations worldwide and instantly view current weather conditions in a clean and user-friendly interface. 🚀 Features:
🔍 Search weather by city name
🌡️ Real-time temperature (°C)
📍 Displays city/location name
🕒 Shows local date and time
🌤️ Live weather condition
🖼️ Dynamic weather icons
⚠️ Error handling for invalid city names
📱 Responsive and minimal UI
🛠️ Technologies Used:
HTML5
CSS3
JavaScript (ES6)
WeatherAPI
Fetch API
